# MW Lead Capture System

A complete PHP + MySQL CRM for capturing and managing leads.

## Features
- Public lead submission form (anyone can add leads)
- Secure admin login (admin / admin123)
- Interactive dashboard with live pie chart
- Full CRUD (Add, Edit, Search, Delete leads)
- Reports with CSV export
- Responsive Bootstrap 5 design

## How to Run Locally
1. Install XAMPP and start Apache + MySQL
2. Create database `mw_leads` and import the SQL
3. Copy folder to `htdocs/mw-lead-capture`
4. Open: `http://localhost/mw-lead-capture`
